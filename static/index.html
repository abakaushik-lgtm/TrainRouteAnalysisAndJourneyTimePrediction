Data Overview & Clean Ingestion

The primary source dataset contains detailed stop-by-stop station timetables. To prepare this data for robust machine learning, the system engineered a complete workflow resolving critical anomalies:

  • Day-Boundary Wrap-Arounds: Resolved sequence time wrap-arounds by tracking HH:MM:SS decreases. Result: 0 negative duration anomalies.
  • Truncated Routes: Audited and filtered out train routes that do not start at cumulative distance 0 (e.g. Train 12978).
  • Circular / Repeat Stations: Preserved valid loop-back stations (e.g. Train 290 at station DSJ) as sequential stop events.
